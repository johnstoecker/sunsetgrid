# sunsetgrid
Outrun synthwave music visualizer

![alt text](http://sunsetgrid.com/sunset_grid.png)


# install
npm install

# webpack dev server:
npm start
 ->> should open localhost:8080 in web browser

# production build:
npm run build
open dist/index.html in browser (you might have to fiddle with CORS)


Controls
--------
spacebar: play/pause
d: spawn a dolphin
